# Korean-Pro-Demo UI 분석

## 전체 구조

- **Thymeleaf 템플릿 엔진** 기반의 반응형 웹 애플리케이션
- 서버 사이드 렌더링 + 정적 리소스(css/js/img)

---

## 템플릿 구조

```
templates/
├── layout/         # 메인/서브 레이아웃, 헤더, 푸터
├── fragment/       # 챗봇 등 컴포넌트
├── sp/             # 음성 평가/처리 페이지
├── file/           # 파일 업로드
├── error/          # 에러 페이지
└── index.html      # 메인 페이지
```

---

## 디자인 시스템

- **컬러 팔레트**: 그라데이션(핑크-오렌지), 다크 그레이, 화이트
- **폰트**: 'Pretendard', 'GmarketSans', 'SBAggroB/M/L' 등 한글 웹폰트
- **반응형**: rem 단위, Flexbox, PC/모바일 분기
- **글래스모피즘**: `backdrop-filter: blur()`로 반투명 효과

---

## 주요 UI 컴포넌트

### 1. 메인 페이지 (index.html)
- 캠프 소개, 일정, 참가 안내, 공지사항(아코디언)
- AOS.js 애니메이션, 원형 강조, 그라데이션 버튼

### 2. 한국어 발음 평가 (sp-ko-demo.html)
- 문제 목록, 녹음 인터페이스, 결과 팝업(점수 시각화)
- 녹음/재녹음, 실시간 평가, 음절/음소별 점수 표시
- RecordRTC.js, sp-ko-demo.js 등 JS로 동작

### 3. 헤더/네비게이션
- Fixed Header, 2-Depth 메뉴, 블러 배경, 둥근 모서리

### 4. 버튼/팝업
- 그라데이션/원형 버튼, 다양한 팝업(alert, 결과, 로딩 등)

---

## 정적 리소스

- **CSS**: reset.css(폰트/초기화), style.css(메인 스타일)
- **JS**: jQuery, AOS, RecordRTC, sp-ko-demo.js 등
- **이미지**: 로고, 아이콘, 메인/서브 이미지

---

## UX/UI 패턴

- 글래스모피즘, 그라데이션, 원형 강조
- 점수 시각화(원형, 색상), 실시간 피드백
- 앵커 네비게이션, 스크롤 애니메이션
- 반응형 레이아웃, 모바일 대응
